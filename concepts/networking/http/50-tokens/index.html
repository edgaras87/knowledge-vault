What a Token Really Is¶

  2. Verify its signature (check it‚Äôs not forged).

  4. Use claims to authorize actions.

All lasting data (orders, messages, progress, preferences) is stored in the database, linked to your user ID. When you send a token, the server uses that ID to look up your data as needed.

Cookies make the web remember you. Tokens make the web verify you.
